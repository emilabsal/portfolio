# Portfolio

Персональный portfolio-проект на Nuxt. Используется как основа для публичной презентации опыта, проектов и frontend-навыков.

## Стек

- Nuxt 4
- Vue 3
- TypeScript
- Nuxt UI
- Tailwind CSS
- ESLint
- pnpm

## Запуск

```bash
pnpm install
pnpm dev
```

## Скрипты

```bash
pnpm dev
pnpm build
pnpm preview
pnpm lint
pnpm typecheck
```
